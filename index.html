How to read your VIN-TAGe™
TIB
Brand Code
THROTTLED IN BOND™. Co-branded releases may show e.g. TIB-HD
26
Bottling Year
Last two digits of the calendar year the whiskey was bottled
FS
Series
FS = Founders Select · CS = Collection · RS = Reserve · SS = Survivor · LS = Limited
G1
Generation Code
Which release generation of the series this barrel belongs to. G1 = First Generation, G2 = Second, and so on.
00001
Barrel Release
Sequential barrel number across all releases. 00001 is the first barrel ever released by THROTTLED IN BOND™
117
Proof
Barrel strength at the time of bottling — nothing added, nothing removed
KY
Origin
State where the whiskey was distilled
6
Age
Years aged in barrel before bottling
E26
Bottling Date
Month letter + year (A=Jan, B=Feb … E=May, etc.)
C74 R18
W0 MB8
Mash Bill
Grain key + percentage. C=Corn · R=Rye · W=Wheat · MB=Malted Barley. A 0% grain is shown but not used. Alternate grains (O=Oat, etc.) replace their slot
